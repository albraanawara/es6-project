# ☕ ES6 Coffee Shop Project

A simple and modern coffee shop web application built using **JavaScript (ES6)**, HTML, and CSS.  
The project displays a list of products with categories like Coffee, Tea, and Beans.

## 🌐 Live Demo

[View Website](https://albraanawara.github.io/es6-project/)

---

##  Features

- Display products dynamically using JavaScript (ES6)
- Filter products by category
- Clean and responsive UI
- Product cards with:
  - Image
  - Name
  - Price
  - Description
- Organized and readable code (Clean Code)

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
